# Project Title

Our project is a web application, that connects trainers to dog owners that need an expert trainer.
In the application, the dog owner chooses what expertise he looking for, and the trainer chooses what expertise he has.
In this way, the application makes the best match between them.


### Prerequisites

Download Visual Studio Code


### Installing

In the treminal:
npm install
npm start

## Running the tests

In the treminal: npm run test

### Break down into end to end tests

The tests that we wrote, checking the database functions and storage.
